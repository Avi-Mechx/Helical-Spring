# 🌀 Helical Spring – AutoCAD 3D Design

![Preview](./HELICAL%20SPRING%20.dwg)

This project presents a 3D CAD model of a **Helical Spring** designed using AutoCAD. It demonstrates spring geometry, coil structure, and mechanical design basics.

---

## 📌 About the Project

A helical spring is used to store mechanical energy and absorb shocks.  
This model represents an extension spring with hooks at both ends.

---

## 🛠️ Tools Used

- AutoCAD (3D Modeling)

---

## ✨ Design Features

- Uniform helical coil  
- Hook ends for attachment  
- Consistent pitch and diameter  
- Smooth 3D structure  

---

## ⚙️ Applications

- Suspension systems  
- Shock absorbers  
- Mechanical assemblies  
- Industrial equipment
